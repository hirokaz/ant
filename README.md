# 🐜 アント (ANT)

通勤中に片手で遊べる、コロニー育成アクションゲーム。

## 起動方法

`index.html` をブラウザで開くだけ。サーバー不要。

## 操作

- **🕹️ 左下のスティック** — 移動（左下半分のどこをタッチしても OK）
- **「攻撃」ボタン** — 隣の敵を攻撃
- **「仲間呼ぶ」ボタン** — 周囲の仲間を集めて連れていく

## ゲームの流れ

1. **小さな餌**は自分で運ぶ（巣の卵部屋へ）
2. **大きな餌**は仲間が必要 — 「仲間呼ぶ」で集めて餌の場所まで連れていけば、**仲間が自動的に運ぶ**
3. 卵部屋に餌を届けると **卵 → 仲間** が増える
4. **敵**が外で襲ってくる — 攻撃するか、仲間を呼んで一緒に戦う、または巣（円の中）に逃げ込む
5. やられたら巣から再出発（仲間はそのまま）
6. **仲間 1000匹** で勝利（100匹増えるごとにフィールドが広がり、新しい地形が解放される）

## 餌の種類 (5種類)

| 種類 | 必要な仲間 | 卵 |
|------|---------|-----|
| 🌾 種 (small) | 1 | 1 |
| 🍃 葉 (medium) | 3 | 4 |
| 🍎 リンゴ (large) | 5 | 7 |
| 🥖 パン (huge) | 8 | 11 |
| 🍰 ケーキ (giant) | 12 | 18 |

コロニーが大きくなるほど大型の餌が出現するようになります。

## 敵の種類 (3種類)

- **🕷 クモ** — 標準的な敵。直接追跡して噛みつく
- **🪲 カブトムシ** — 高HP・高火力。チャージしてから直線突進（赤い破線が予兆）
- **🐝 ハチ** — 高速・低HP。周回しながら定期的にダイブ攻撃

地形によっては**強化個体**（赤いオーラ）が出現し、HP/攻撃力が高い。

## 🛡 巣レイド (Nest Raid)

仲間が10匹以上のとき、たまに敵が**巣を直接襲ってくる**。レイダーは **赤いリング** で識別でき、**巣の中にも侵入する**。卵を破壊されると孵化を待っていた仲間が消えてしまう。

- 画面外の最寄りレイダーには **⚠️ 矢印インジケータ** で位置を表示
- 巣にいる仲間は自動で応戦し、戦闘中の仲間が倒れると周囲に**警報**が伝播
- 全レイダーを撃退するとボーナスでハートが出現

## 🌍 地形システム

外には7種類の地形がある。地形毎に歩きやすさ・餌/敵の出現量・特殊効果が異なる。

| 地形 | 効果 |
|------|------|
| 🟩 草原 | 標準 |
| 🟦 池 | 大幅減速・HPじわじわ減少 (ハチは飛んで無視) |
| 🟨 砂 | やや減速。クモが多い |
| 🟫 泥 | 大幅減速・HPじわじわ減少。カブトムシ多め (強化) |
| 🌸 花畑 | 餌が豊富。HPが微回復。ハチが多め |
| 🍂 落ち葉 | やや減速。クモが隠れる |
| ⬜ コンクリ | 歩きやすい。**強い**敵が多い。たまに大型餌 (人の落とし物) |

## 🌍 フィールド拡張

最初は**草エリアだけ**。仲間が **50匹増える毎** にフィールドが東に広がり、新しいバイオーム帯が解放される。各エリアは初期の約25%サイズ・80%が該当地形。

| 仲間数 | 解放されるエリア / 追加される要素 |
|------|--------------------------------|
| 50  | 🟫 泥エリア + カブトムシ出現 + 🍎 大型餌 |
| 100 | 🟦 池エリア + ハチ出現 + 🥖 特大餌 |
| 150 | 🌸 花畑エリア + 🍰 超特大餌 |
| 200 | 🍂 落ち葉エリア |
| 250 | 🟨 砂エリア |
| 300 | ⬜ コンクリエリア (強敵&大型餌) |
| 350+ | 各エリアが順次拡張 (50匹毎) |

各エリア解放時に🎁ハート1個ボーナス。1000匹でクリア。

## 回復アイテム

**💖 ハート** が時々現れる。触れるとプレイヤー HP +40、近くの仲間も少し回復。

## 仲間の挙動

- 「仲間呼ぶ」で周囲 200px の仲間が**追従**状態になる
- 大型の餌の近くに来ると、必要数の仲間が**自動的に運搬**を開始
- 運ばない仲間は引き続きプレイヤーを追従（運搬中も護衛できる）
- 餌を巣に届けたら、運んだ仲間も追従に戻って次の餌を探せる

## ヒント

- 序盤は小さい餌で仲間を増やす
- 「仲間呼ぶ」を使って大型の餌に挑戦するタイミングを見極めよう
- HPが低いときはハートを探そう
- カブトムシのチャージは赤い破線で予告 — 横にステップして避けよう
- ハチは離れて回避、近づいてきた瞬間に攻撃
- 巣の中は基本的に安全だが、**レイダー（赤いリング）は侵入してくる**
- 池・泥はじわじわHPが減るので長居しない
- 花畑は餌が多くHPも回復するので休憩スポットに最適
